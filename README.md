#Mind Connect

## GETTING STARTED
1. Clone the project
 
   ```
   git clone https://github.com/Ayine-nongre/MindConnect.git
   ```
2. Change to project directory

    ```
    cd MindConnect/
    ```
3. Set up enviroment variables in env file
   
4. Install packages used in project

   ```
   npm install
   ```
5. Start server

    ```
    npx expo start
    ```
